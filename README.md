# AeroRescue Swarm: Multi-Agent Disaster Logistics System

Built for the **Google x Kaggle AI Agents Intensive Capstone**, this project chose the "Agents for Good" track.

### 📝 Project Rationale & Overview
AeroRescue Swarm addresses the critical bottleneck of delayed manual triage and logistics in disaster zones. The decentralized AI architecture coordinates three distinct agents—**Recon**, **Medical**, and **Commander**—to process real-time simulated telemetry and prioritize autonomous drone-based supply drops to high-severity sectors instantly.

### 🎯 Key Tech Stack
*   Multi-Agent System Orchestration
*   Agent Development Kit (ADK)
*   Python
*   Simulated FastMCP Mock Server for telemetry input.
